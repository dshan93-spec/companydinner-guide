# Company Dinner Guide

## 회식 장소 변경 안내

기존 회식 장소를 **기흥 인근 횟집**으로 변경합니다.

## 선정 장소

**청해수산 기흥구청 인근점**  
- 추천 이유: 기흥/기흥구청 인근에서 접근성이 좋고, 회식 메뉴로 무난한 활어회·모둠회 중심의 장소로 적합합니다.
- 회식 성격: 팀 회식, 소규모/중규모 단체 식사에 적합
- 참석 인원: **9명**
- 메뉴 방향: 모둠회 + 매운탕 + 식사류
- 예약 필요 사항: 인원 9명 기준, 룸/단체석 가능 여부, 1인 예산 확인

> 참고: 실제 예약 전에는 영업 여부, 좌석 가능 여부, 주차 가능 여부를 전화로 최종 확인하는 것을 권장합니다.
